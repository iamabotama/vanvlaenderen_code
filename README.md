# vanvlaenderen_code

**Code repository for [vanvlaenderen.org](https://vanvlaenderen.org)**

This repository contains all development tooling, scripts, automation, configuration, and supporting code for the vanvlaenderen.org project. The live website source lives in the companion repository [`vanvlaenderen.org`](https://github.com/iamabotama/vanvlaenderen.org).

---

## Repository Structure

```
vanvlaenderen_code/
├── scripts/          # Automation and utility scripts
├── config/           # Shared configuration files
├── docs/             # Developer documentation and notes
└── .github/
    ├── ISSUE_TEMPLATE/
    └── workflows/    # CI/CD GitHub Actions workflows
```

## Related Repositories

| Repository | Purpose |
|---|---|
| [`iamabotama/vanvlaenderen.org`](https://github.com/iamabotama/vanvlaenderen.org) | Live website source (Vite + React, GitHub Pages) |
| [`iamabotama/vanvlaenderen_code`](https://github.com/iamabotama/vanvlaenderen_code) | This repo — tooling, scripts, and dev resources |

## Getting Started

Clone this repository alongside the website repo:

```bash
git clone https://github.com/iamabotama/vanvlaenderen_code.git
git clone https://github.com/iamabotama/vanvlaenderen.org.git
```
